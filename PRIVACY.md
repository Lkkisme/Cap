# Privacy Policy

Cap Chinese Edition is a screen recording application. It can access screen content, microphone audio, system audio, and camera video when the user chooses to record.

## Local Recording

Recordings may be created and processed on the user's computer.

The application may store local settings, recording metadata, temporary recording data, logs, and exported files on the user's computer.

## Uploads and Network Use

The application can connect to Cap web services for account sign-in, organization features, upload, sharing, updates, and related cloud functionality.

Recorded content is not intentionally uploaded unless the user signs in and chooses a workflow that uploads or shares the recording.

Network features may contact configured Cap service endpoints and supporting infrastructure needed for authentication, upload, sharing, updates, or application functionality.

## Sensitive Permissions

The application may request operating system permissions for:

- Screen recording
- Microphone access
- Camera access
- File access for importing, exporting, and saving recordings

These permissions are used to provide recording and editing functionality.

## Chinese Localization Fork

This repository is a Chinese-localized fork of [CapSoftware/Cap](https://github.com/CapSoftware/Cap). Upstream service behavior and policies may also apply when the application connects to upstream or configured Cap services.

## No Security Bypass Purpose

The application is not intended to bypass endpoint security, access controls, or organization policies.

Users should follow the rules of their organization before installing or running the application on managed computers.
